concepto descricion
tabla en thml Una tabla en HTML se crea utilizando la etiqueta, que contiene filas y celdas para organizar datos de manera estructurada.
etiqueta table Es el contenedor principal de la tabla. Dentro de esta etiqueta se definen las filas y celdas
atributos table Se pueden usar atributos como border, cellpadding, y cellspacing para definir el estilo de la tabla.
etiqueta tr Define una fila en la tabla. Cada fila puede contener varias celdas.
Atributo tr Se pueden usar atributos como bgcolor para cambiar el color de fondo de la fila.
Etiqueta td y th (td) se utiliza para las celdas de datos, mientras que th se utiliza para las celdas de encabezado.
atributos de td y th Atributos como colspan y rowspan permiten que una celda ocupe más de una columna o fila, respectivamente.
tablas avanzadas Se pueden utilizar elementos como <.thead>, <.tbody>, y <.tfoot> para estructurar mejor la tabla y mejorar la accesibilidad.
etiqueta thead Define el encabezado de la tabla, que puede contener una o más filas de encabezado.
Etiqueta tfoot Define el pie de la tabla, que puede contener información de resumen o totales.
etiqueta tbody Contiene el cuerpo principal de la tabla, donde se encuentran las celdas de datos.
formulario en html Un formulario se utiliza para recopilar datos del usuario. Se crea utilizando la etiqueta <.form>.
etiqueta from Es el contenedor principal para los elementos del formulario.
etiqueta input Se utiliza para crear campos de entrada, como texto, contraseñas, y botones.
etiqueta label Mejora la accesibilidad al asociar un texto descriptivo con un control de formulario.
etiqueta select Crea un menú desplegable que permite al usuario seleccionar una opción de una lista.
etiqueta textarea Permite la entrada de texto en múltiples líneas.
etiqueta fieldset legend Agrupa controles relacionados dentro de un formulario, mejorando la organización y la accesibilidad.
etiqueta meter Representa un valor escalar dentro de un rango conocido, como un medidor.
etiqueta progress Indica el progreso de una tarea en curso, como una barra de carga.
etiqueta datalist Proporciona una lista de opciones predefinidas para un campo de entrada.
etiqueta summary Se utiliza para crear un resumen que puede ser expandido o colapsado, útil en formularios complejos.

Tema 2. Define los elementos que componen la estructuración de una página web:

1. Contenedor (wrapper)

El contenedor, o "wrapper", es un elemento que envuelve todos los demás componentes de la página. Su función principal es proporcionar un marco que centraliza el contenido y establece un ancho máximo, ayudando a mantener la coherencia visual en diferentes tamaños de pantalla.

2. Cabecera (header)

La cabecera es la parte superior de la página y suele incluir el logotipo, el título del sitio y el menú de navegación. Es crucial porque es lo primero que los usuarios ven al ingresar a la página, y debe ser clara y atractiva para facilitar la navegación

3. Contenido (content)

El contenido es el área principal de la página donde se presenta la información más relevante, como textos, imágenes, videos y otros elementos multimedia. Este espacio debe ser organizado de manera que los usuarios puedan encontrar fácilmente lo que buscan.

4. Menú (menu)

El menú es un elemento de navegación que permite a los usuarios acceder a diferentes secciones del sitio web. Generalmente se encuentra en la cabecera, pero también puede aparecer en el pie de página o en la barra lateral. Un menú bien estructurado mejora la experiencia del usuario al facilitar la navegación

5. Pie (footer)

El pie de página es la sección que se encuentra al final de la página. Suele contener información adicional como enlaces a políticas de privacidad, términos de uso, información de contacto y enlaces a redes sociales

6. Lateral (sidebar)

La barra lateral es un área que se encuentra a un lado del contenido principal. Puede contener widgets, enlaces a artículos relacionados, anuncios o cualquier otro contenido adicional que complemente la información principal.

7. Etiqueta div

La etiqueta <.div> es un elemento de HTML que se utiliza para agrupar otros elementos y aplicar estilos a ellos. Es fundamental para la estructuración de la página, ya que permite dividir el contenido en secciones lógicas y aplicar estilos CSS de manera efectiva
